CSS

CSS = Cascading Style Sheets CSS is used to beautify our html document and makes our web page more attractive. we can make html document more attractive by using CSS properties like background, border, fonts, color, display, postion and so many. CSS is used for beautification of html document.

CSS's first priority is inline css.

We can apply/adopt CSS in three ways.

  1. INLINE
    • INLINE CSS
  2. INTERNAL
    • INTERNAL CSS
  3. EXTERNAL
